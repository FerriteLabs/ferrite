# Ferrite Graph

Graph data model and traversal operations for Ferrite.

Part of the [Ferrite](https://github.com/ferritelabs/ferrite) workspace.

## Modules

- `graph` — Property graph database with traversal, pattern matching, and shortest-path algorithms

## Usage

Add to your `Cargo.toml`:

```toml
[dependencies]
ferrite-graph = { git = "https://github.com/ferritelabs/ferrite" }
```
