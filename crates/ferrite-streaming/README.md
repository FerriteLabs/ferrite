# Ferrite Streaming

Event streaming, change data capture (CDC), and data pipelines for Ferrite.

Part of the [Ferrite](https://github.com/ferritelabs/ferrite) workspace.

## Modules

- `streaming` — Event streaming and consumer groups
- `cdc` — Change data capture for external systems
- `pipeline` — Data processing pipelines

## Usage

Add to your `Cargo.toml`:

```toml
[dependencies]
ferrite-streaming = { git = "https://github.com/ferritelabs/ferrite" }
```
