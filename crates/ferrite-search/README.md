# Ferrite Search

Full-text search engine, auto-indexing, and query routing for Ferrite.

Part of the [Ferrite](https://github.com/ferritelabs/ferrite) workspace.

## Modules

- `search` — Full-text search with indexing and query language
- `autoindex` — Automatic index creation and management
- `routing` — Query routing and index selection
- `schema` — Schema validation and evolution

## Usage

Add to your `Cargo.toml`:

```toml
[dependencies]
ferrite-search = { git = "https://github.com/ferritelabs/ferrite" }
```
