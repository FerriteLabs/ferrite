# Ferrite K8s

Kubernetes operator and integration for Ferrite.

Part of the [Ferrite](https://github.com/ferritelabs/ferrite) workspace.

## Modules

- `k8s` — CRD definitions, controller, reconciler, and operator for Kubernetes-native Ferrite deployment

## Usage

Add to your `Cargo.toml`:

```toml
[dependencies]
ferrite-k8s = { git = "https://github.com/ferritelabs/ferrite" }
```
