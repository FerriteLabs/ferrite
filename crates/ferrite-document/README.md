# Ferrite Document

JSON document store with path queries for Ferrite.

Part of the [Ferrite](https://github.com/ferritelabs/ferrite) workspace.

## Modules

- `document` — MongoDB-compatible document store with indexing, aggregation, and validation

## Usage

Add to your `Cargo.toml`:

```toml
[dependencies]
ferrite-document = { git = "https://github.com/ferritelabs/ferrite" }
```
