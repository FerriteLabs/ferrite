# Ferrite Studio

Web-based management UI and interactive playground for Ferrite.

Part of the [Ferrite](https://github.com/ferritelabs/ferrite) workspace.

## Modules

- `studio` — Web-based management dashboard
- `playground` — Interactive query playground
- `insights` — Automated performance insights

## Usage

Add to your `Cargo.toml`:

```toml
[dependencies]
ferrite-studio = { git = "https://github.com/ferritelabs/ferrite" }
```
