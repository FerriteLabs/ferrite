# Ferrite TimeSeries

Time-series data ingestion, downsampling, and retention for Ferrite.

Part of the [Ferrite](https://github.com/ferritelabs/ferrite) workspace.

## Modules

- `timeseries` — Time-series engine with aggregation, compression, and retention policies

## Usage

Add to your `Cargo.toml`:

```toml
[dependencies]
ferrite-timeseries = { git = "https://github.com/ferritelabs/ferrite" }
```
